¿Qué es y que función cumple el CSS?

CSS es una función que se agrega a HTML para facilitar a los usuarios el control de las páginas web.

¿Cómo se incorpora el CSS dentro del HTML?

El CSS se incorpora como un método común para adjuntar reglas CSS a un documento HTML.

¿Cuál es la anatomía de una regla CSS?

Selector, llave de apertura, propiedad, valor, llave de cierre.

¿Cuáles son los tipos de selectores que puede utilizar CSS?

Los tipos de selectores son: Selector universal, Selector etiqueta, Selector clase, Selector identificador, Selector descendiente, Combinación de selectores, Selector de hijo, Selector adyacente.

¿Qué es javascript? ¿Podrías definirlo como un lenguaje de programación? ¿De qué tipo? Justificar.

Javascript es un lenguaje de programación o secuencia de comandos, de tipo numérica y alfanumérica.

¿A qué se llama DOM y cual es su función?

DOM es una interfaz de programación para los documentos HTML y XML, donde facilita una representación estructurada de un documento y define la manera en que los programas puedan acceder con el fin de modificar su estructura, estilo y contenido.
